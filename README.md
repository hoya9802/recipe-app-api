## 🗂️움짤 저장소

### 👉프로젝트 소개 <br>
- 가지고 있는 움짤들은 많지만 이것들을 공유할 수 없을 때
- 다른 사람들이 가지고 있는 움짤들이 궁금할 때
- 나랑 비슷한 관심사를 가지고 있는 사람들을 구독하고 싶을 때
- 내가 가진 움짤들로 유저들과 이야기를 나누고 싶을 때

💡가지고 있는 움짤들을 공유할 수 있는 사이트, 움짤 저장소 입니다! 🥳
